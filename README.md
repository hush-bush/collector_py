# Сборщик средств для сети Base

Программа для автоматического сбора ERC-20 токенов с нескольких кошельков в сети Base на один целевой кошелек.

## Описание

Этот скрипт позволяет автоматически собрать токены с нескольких кошельков Ethereum в сети Base и отправить их на один целевой адрес. Программа поддерживает:

- Автоматический сбор токенов ERC-20
- Работу с несколькими кошельками одновременно
- Переключение между RPC нодами при сбоях
- Настраиваемую задержку между транзакциями
- Подробное логирование процесса

## Требования

- Python 3.7+
- web3.py

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/YOUR_USERNAME/collector_py.git
cd collector_py
```

2. Установите зависимости:
```bash
pip install web3
```

## Настройка

1. Скопируйте `.env.example` в `.env` (если есть) или создайте файл `.env` со списком приватных ключей:
```
PRIVATE_KEY_1
PRIVATE_KEY_2
PRIVATE_KEY_3
```

2. Отредактируйте `config.py`:
   - `RPC_URL` - адрес RPC ноды сети Base
   - `RECIPIENT_ADDRESS` - адрес кошелька-получателя
   - `TOKEN_ADDRESS` - адрес токена ERC-20 для сбора
   - `GAS_PRICE` - цена газа в gwei
   - `DELAY` - задержка между транзакциями в миллисекундах

## Использование

Запустите скрипт:
```bash
python collector.py
```

## Важно

- **НЕ ПУБЛИКУЙТЕ** файл `.env` с приватными ключами в публичный репозиторий
- Убедитесь, что на каждом кошельке есть достаточно BASE для оплаты газа
- Рекомендуется сначала протестировать на тестовой сети
